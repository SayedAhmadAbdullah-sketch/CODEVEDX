What is Phishing?

Red Flags

  • Suspicious URL - Not the legitimate domain
  • Generic Greeting - "Dear Customer" not your name
  • False Urgency - "Account suspended in 24 hours!"
  • Credential Request - Asking for passwords via email
  • Grammar Errors - Subtle spelling mistakes
  • Mismatched Links - Display link vs actual destination differ

Types of Phishing

1. Email Phishing

Mass emails impersonating companies.

2. Spear Phishing

Targeted attacks using personal info.

3. Whaling

Targeting executives (CEOs, CFOs).

4. Clone Phishing

Exact copies of legitimate emails.

5. Smishing

Phishing via SMS text messages.

6. Vishing

Phishing via phone calls.

Protection Checklist

  • Always check the URL before entering credentials
  • Look for HTTPS padlock icon
  • Never click suspicious links - go directly to the website
  • Enable Two-Factor Authentication (2FA) on all accounts
  • Use a password manager
  • Verify sender's email address
  • Be suspicious of urgent/threatening language
  • Report suspicious emails to your IT team
  • Keep browser and antivirus updated
  • Use email filtering and anti-phishing tools

Technical Detection

URL Analysis: Check domain registration, WHOIS, DNS records.
Email Headers: Examine SPF, DKIM, DMARC records.
SSL Certificate: Verify certificate issuer and validity.
Remember: This was educational. Real phishing causes identity theft and financial loss.
